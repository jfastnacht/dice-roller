# Dice-Roller

> A simple pen & paper dice roller written in PHP.

## Installation

Clone repository and run [Composer](https://getcomposer.org/) installation:

`composer install`

## Usage

Within the directory run:

`php application.php dice:roll 1d20+2`

to roll a d20 dice +2.
